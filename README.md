Aplikasi CampusInfo merupakan mini project yang dikembangkan dengan kotlin. Aplikasi ini memiliki berbagai fitur utama seperti Button Navigation (mempermudah pengguna berpindah halaman),
RecyclerView & ListAdapter (penampil daftar pengumuman kepada pengguna), Navigation ke Detail (fitur interaktif dari navigasi ke home lalu ke halaman detail), Livecycle & ViewBinding
(mengurangi kebocoran memori/data hilang saat rotasi), ViewModel (Tempat penyimpanan data), dan terakhir komunikasi antar fragment menggunakan Shared ViewModel sehingga data dari home
dapat ditampilkan di profile. Aplikasi ini juga memiliki fitur khusus seperti mendukung dark mode, memiliki filter pengumuman, adanya animasi navigasi/transisi, pengguna dapat
mengubah profil dan menyimpannya, lalu terakhir adalah menampilkan header berdasarkan hari.
